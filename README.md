# Математические пакеты программ. Лабораторная работа №3: Однофакторный регрессионный анализ в Python

## Цель работы
Знакомство с методами однофакторного регрессионного анализа в Python, с использованием библиотеки scikit-learn.

## Задание на лабораторную работу
1. Вычислить номер варианта как остаток от деления **суммы** порядкового номера студента по списку в журнале на количество вариантов заданий. Если остаток равен нулю, выбрать последнее задание.  
**Nвар = Nв_списке mod K**,  
где Nвар - искомый номер варианта, Nв_списке - порядковый номер студента по списку в журнале, K - количество вариантов заданий на данную лабораторную работу.
2. Выбрать экспериментальные данные в соотвествии с вариантом.
3. Построить уравнение регрессии в виде линейного алгебраического полинома от двух переменных.
4. Проверить адекватность построенного уравнения регрессии экспериментальным данным по критерию Фишера при уровне значимости α = 0,05.
5. В случае неадекватности линейного уравнения построить уравнение регрессии в виде неполного квадратичного полинома от двух переменных:

6. Проверить адекватность построенного уравнения регрессии экспериментальным данным по критерию Фишера при уровне значимости α = 0,05.
7. Выполнить селекцию факторов по критерию Стьюдента при таком же уровне значимости.
8. Повторно проверить адекватность уравнения регрессии после исключения незначимых факторов.
